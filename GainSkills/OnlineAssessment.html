1. Aptitude & Logical Reasoning (For All Companies) Most MNCs and IT companies include an aptitude section in their online tests. Common topics include:

a) Quantitative Aptitude

✔ Number System & HCF/LCM
✔ Profit & Loss
✔ Time, Speed & Distance
✔ Time & Work
✔ Permutations & Combinations
✔ Probability
✔ Ratio & Proportion
✔ Data Interpretation (Pie Charts, Bar Graphs, Line Graphs, Tables)
✔ Simple & Compound Interest
✔ Averages, Percentages, and Mixtures
✔ Logarithms and Surds

b) Logical Reasoning & Analytical Thinking

✔ Coding-Decoding
✔ Blood Relations
✔ Seating Arrangement (Linear, Circular)
✔ Syllogisms
✔ Data Sufficiency
✔ Statement & Assumption
✔ Cause & Effect
✔ Puzzle Solving
✔ Directions & Distance
✔ Odd-One-Out

c) Verbal Ability & English Proficiency

✔ Reading Comprehension
✔ Sentence Completion & Correction
✔ Synonyms & Antonyms
✔ Spotting Errors
✔ Cloze Test
✔ Para Jumbles
✔ Active & Passive Voice
✔ Direct & Indirect Speech

2. Programming & Technical Topics (For IT Companies & Tech Roles) For software and IT roles, coding and technical knowledge play a significant role in the assessment.

a) Programming Languages

✔ C, C++ Basics
✔ Java
✔ Python
✔ SQL (Database Queries)
✔ Object-Oriented Programming (OOP)

b) Data Structures & Algorithms (DSA)

✔ Arrays & Strings
✔ Linked Lists
✔ Stacks & Queues
✔ Recursion
✔ Searching & Sorting (Binary Search, Quick Sort, Merge Sort)
✔ Trees & Graphs (DFS, BFS, Binary Tree, BST)
✔ Hashing
✔ Dynamic Programming (Knapsack, Fibonacci, Longest Common Subsequence)
✔ Bit Manipulation

c) Database Management System (DBMS)

✔ SQL Queries (Joins, Subqueries, Normalization)
✔ ACID Properties
✔ Transactions & Concurrency

d) Operating System (OS)

✔ Process Management (Threads, Scheduling)
✔ Memory Management (Paging, Segmentation)
✔ Deadlocks
✔ File Systems

e) Computer Networks

✔ TCP/IP Model & OSI Model
✔ Network Protocols (HTTP, FTP, DNS)
✔ IP Addressing & Subnetting

f) Object-Oriented Programming (OOPs)

✔ Classes & Objects
✔ Inheritance
✔ Polymorphism
✔ Encapsulation & Abstraction

3. Coding & Problem-Solving Section Most IT companies (e.g., TCS, Infosys, Wipro, Cognizant, Capgemini, Amazon, Google, Microsoft) have a coding test in their online assessment.

✔ Basic Programming Challenges (Loops, Conditions, Functions)
✔ Competitive Programming (LeetCode, CodeChef, HackerRank practice)
✔ Pattern Printing Problems
✔ String Manipulation (Anagrams, Palindrome)
✔ Array Manipulations (Merging, Rotation, Sliding Window)
✔ Graph & Tree Problems

4. Domain-Specific Questions (For Specialized Roles) For non-software roles like Data Science, AI/ML, Cybersecurity, and Cloud Computing, additional topics include:

a) Data Science & Machine Learning

✔ Probability & Statistics
✔ Regression & Classification
✔ Neural Networks & Deep Learning
✔ Data Visualization (Pandas, Matplotlib)

b) Cybersecurity

✔ Encryption & Decryption
✔ Authentication & Authorization
✔ Network Security Protocols

c) Cloud Computing

✔ AWS, Azure, Google Cloud Basics
✔ Virtualization & Containerization (Docker, Kubernetes)
